Avantrip.com

UI, UX

The project
Avantrip’s hotel section was a new product for the company. Being one of the few e-commerce sites at the time in Argentina, the stakeholders wanted the release to be as soon as possible, specially because the market was growing and new competitors were starting to emerge.

The challenge
At that time the mobile users in Argentina were not that many so most the effort was put to finish the desktop version and the responsive view. But soon we realized that an adaptive version was needed to give the future mobile users a better experience, so that’s where i come in. With a very short time to work, using an agile approach, we managed to build a first version and create the foundations for this section.

Role:
UX, UI designer
